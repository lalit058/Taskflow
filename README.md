# Taskflow - Full-Stack Task Management System

A MERN stack application built for the Codveda Internship. 
Features include JWT Authentication, CRUD operations, and Socket.io integration.

## 🚀 Features
- **Secure Auth**: Login/Register with JWT stored in Session Storage.
- **Task CRUD**: Create, Read, Update, and Delete tasks.
- **Real-time**: Live updates using Socket.io.
- **API**: Hybrid REST and GraphQL architecture.

## 🛠️ Tech Stack
- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **Real-time**: Socket.io
- **Query Language**: GraphQL (Apollo Server)

## 🏃 Getting Started
1. Clone the repo: `git clone https://github.com/lalit058/Taskflow.git`
2. Install dependencies for both folders: `npm install`
3. Set up your `.env` file (JWT_SECRET, MONGODB_URI).
4. Run the server: `npm run dev`
